Bienvenue dans notre démonstrateur d'application web conçu pour illustrer l'intégration harmonieuse de Flask, SQLite3, Pandas et Matplotlib. Cette application a été développée dans le but de présenter une approche efficace pour visualiser et analyser des données provenant d'une base de données locale.

Fonctionnalités principales

  1. Flask : L'application repose sur Flask, un framework web léger et puissant pour Python. Flask facilite la création d'applications web robustes et extensibles.
  2. SQLite3 : La gestion des données est assurée par SQLite3, une base de données relationnelle légère et autonome. Nous exploitons la simplicité et la portabilité de SQLite3 pour stocker et récupérer les informations nécessaires.
  3. Pandas : Pandas, la bibliothèque d'analyse de données de Python, est utilisée pour manipuler et structurer les données extraites de notre base SQLite3. Les fonctionnalités puissantes de Pandas facilitent l'exploration et la transformation des données.
  4. Matplotlib : Pour une visualisation efficace, nous avons intégré Matplotlib, une bibliothèque graphique en 2D qui génère des graphiques. Matplotlib nous permet de créer des représentations visuelles des données, y compris des boîtes à moustaches pour une analyse détaillée.

Comment utiliser l'application

  1. Accès à la Base de données : L'application se connecte à une base de données SQLite3 locale. Assurez-vous d'avoir la base de données correctement configurée et remplie avec les données nécessaires.
  2. Visualiser le contenu des tables : L'utilisation de requêtes SQLite, déclechées avec Pandas, qui renvoient le contenu des tables
  3. Sélectionner des données : Utiliser la technique des jointures SQL afin de rechercher des données
  4. Visualisation des statistiques : Explorez les statistiques de chaque classe, y compris la moyenne, l'écart-type, le minimum et le maximum des notes.
  5. Représentations graphiques : Découvrez la distribution des notes ou encore des boîtes à moustaches générées par Matplotlib pour une représentation visuelle complète des données.
